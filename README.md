# ekutnik.github.io

**Landing page for an independent Ghost research project.**

Built to explore the onboarding gap between signing up for Ghost and feeling confident using it. The page presents two AI-powered tools built to close that gap, grounded in research from YouTube creator tutorials, community forums, and market signals.

## Live site
https://ekutnik.github.io

## Tools featured

**Ghost Onboarding Assistant** — generates a personalised 30-day action plan for independent creators starting on Ghost. Takes niche, audience size, monetisation goal, and technical level as inputs.
→ https://ekutnik.github.io/ghost-onboarding-assistant/

**Ghost Fit Checker** — answers "Is Ghost right for me?" with a frank, personalised verdict covering plans, limitations, migration considerations, and what to do next.
→ https://ekutnik.github.io/ghost-fit-checker/

## Research basis
- YouTube creator tutorials (Liz Rowe, David Utke) — comment sections surface recurring confusion points
- Ghost community forum — excellent resource but hard to discover for first-time visitors
- Synaps Media — third-party business built to simplify Ghost, a market proxy signal that the onboarding gap is real
